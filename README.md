# TI2_ex4
# Exercício 4 - Computação em Nuvem e Serviços Cognitivos

Este repositório contém os entregáveis do Exercício 4 da disciplina de Computação em Nuvem e Serviços Cognitivos.

## Objetivo

O objetivo da atividade foi realizar a ativação de uma conta Microsoft Azure for Students, criar um banco de dados PostgreSQL no Azure, inserir dados em uma tabela e executar consultas SQL utilizando a plataforma de computação em nuvem da Microsoft.

## Conteúdo da pasta

* Imagem do Portal Azure mostrando a conta vinculada e o recurso PostgreSQL criado.
* Imagem da conexão ativa com o banco PostgreSQL.
* Execução de comandos SQL de criação de tabela, inserção de dados e consulta utilizando `SELECT`.

## Tecnologias utilizadas

* Microsoft Azure
* Azure Database for PostgreSQL
* PostgreSQL
* PowerShell / psql

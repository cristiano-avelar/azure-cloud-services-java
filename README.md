# Exercício 4 — Microsoft Azure e Serviços Cognitivos

Projeto desenvolvido para a disciplina de Tecnologias da Informação II da PUC Minas, com o objetivo de explorar recursos de computação em nuvem utilizando a plataforma Microsoft Azure.

O trabalho aborda:
- Criação e configuração de conta Azure for Students
- Criação e utilização de banco de dados PostgreSQL em nuvem
- Consumo de Serviços Cognitivos da Microsoft Azure utilizando Java

---

## Objetivos do Projeto

Este exercício teve como finalidade proporcionar experiência prática com serviços de computação em nuvem e inteligência artificial disponibilizados pela Microsoft Azure.

Durante o desenvolvimento foram realizados:
- Ativação da conta acadêmica Azure
- Criação de recursos em nuvem
- Configuração de banco PostgreSQL
- Execução de consultas SQL
- Integração de aplicações Java com APIs cognitivas do Azure

---

## Tecnologias Utilizadas

- Java
- Microsoft Azure
- Azure Cognitive Services
- PostgreSQL
- pgAdmin
- Git e GitHub
